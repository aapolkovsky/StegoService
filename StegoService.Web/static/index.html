Стеганографический Алгоритм Бенхама

Инструкция

Внедрение текста в изображение

  • Нажав на кнопку "Выбирете файл" выберете bmp/png контейнер.
  • В текстовом поле введите текст для внедрения.
  • Нажмите на кнопку "Внедрить".
  • Дождитесь загрузки изображения.

Извлечения текста из изображения

  • Нажав на кнопку "Выбирете файл" выберете bmp/png контейнер с внедрённым текстом.
  • Нажмите на кнопку "Извлечь".
  • Дождитесь пока в текстовом поле не появится результат.
